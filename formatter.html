Known issues (there are probably a lot of unknown one too):
1. If you leave a box blank in a CRONUSCalc input template spreadsheet (e.g., the 'pressure' box) and tabs don't properly carry through cutting and pasting, you may get weird results. Easiest to make sure there are no empty boxes.
2. You can't use the 'pressure' option in either v3 or CRONUSCalc input, because CREp input format doesn't support that.
3. Transfer of multiple nuclide measurements on the same sample between formats may not be perfect. Tough to handle all possible contingencies here.
4. If something doesn't work, send me the input data and tell me what the problem is.
